# Get Cross-Chain Transfer

Retrieve the current status and details of a cross-chain transfer.

## Method Signature

```typescript
align.crossChain.getTransfer(transferId: string): Promise<CrossChainTransfer>
```

## Example

```typescript
const transfer = await align.crossChain.getTransfer(
  "xc_123e4567-e89b-12d3-a456-426614174000"
);

console.log(`Status: ${transfer.status}`);
console.log(
  `Source: ${transfer.source_network} â†’ ${transfer.destination_network}`
);
```
