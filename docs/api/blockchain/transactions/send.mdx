# Send Transaction

Send native tokens or execute contract calls on EVM-compatible blockchains.

## Method Signature

```typescript
align.blockchain.transactions.send(
  tx: TransactionRequest
): Promise<Transaction>
```

## Parameters

<ParamField body="from" type="string" required>
  Sender wallet address
</ParamField>

<ParamField body="to" type="string" required>
  Recipient address
</ParamField>

<ParamField body="value" type="string">
  Amount in wei (native token)
</ParamField>

<ParamField body="data" type="string">
  Encoded contract call data
</ParamField>

<ParamField body="privateKey" type="string" required>
  Private key for signing
</ParamField>

## Example

```typescript
import { ethers } from "ethers";

// Send 0.1 ETH
const tx = await align.blockchain.transactions.send({
  from: "0x...",
  to: "0x...",
  value: ethers.parseEther("0.1").toString(),
  privateKey: process.env.PRIVATE_KEY!,
});

console.log(`TX Hash: ${tx.hash}`);
console.log(`Status: ${tx.status}`);
```

<Warning>
  Never expose private keys in client-side code. Use server-side signing or
  hardware wallets.
</Warning>
