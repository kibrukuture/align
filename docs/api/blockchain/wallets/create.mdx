# Create Blockchain Wallet

Create a new managed blockchain wallet with optional encryption. Supports HD wallet generation from mnemonic phrases.

## Method Signature

```typescript
align.blockchain.wallets.create(
  options: WalletCreationOptions
): Promise<Wallet | EncryptedWallet>
```

## Parameters

<ParamField body="network" type="string" required>
  Blockchain network for the wallet
</ParamField>

<ParamField body="password" type="string">
  Optional password for wallet encryption
</ParamField>

<ParamField body="mnemonic" type="string">
  Optional 12/24-word mnemonic for deterministic generation
</ParamField>

## Examples

### Create New Wallet

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    import Align from "@tolbel/align";

    const align = new Align({
      apiKey: process.env.ALIGN_API_KEY!,
      environment: "sandbox",
    });

    // Create a new Polygon wallet
    const wallet = await align.blockchain.wallets.create({
      network: "polygon",
    });

    console.log(`Address: ${wallet.address}`);
    console.log(`Mnemonic: ${wallet.mnemonic}`);
    // Store mnemonic securely - cannot be recovered!
    ```

  </Tab>
  <Tab title="JavaScript">
    ```javascript
    const wallet = await align.blockchain.wallets.create({
      network: "polygon",
    });

    console.log("Address:", wallet.address);
    ```

  </Tab>
</Tabs>

### Create Encrypted Wallet

```typescript
const encryptedWallet = await align.blockchain.wallets.create({
  network: "ethereum",
  password: "secure-password",
});

console.log(`Address: ${encryptedWallet.address}`);
console.log(`Encrypted JSON: ${encryptedWallet.encryptedJson}`);
// Store encryptedJson securely
```

### Import from Mnemonic

```typescript
const wallet = await align.blockchain.wallets.create({
  network: "polygon",
  mnemonic: "word1 word2 word3 ... word12",
});
```

<Warning>
  **Store mnemonics and private keys securely!** They cannot be recovered if
  lost.
</Warning>
