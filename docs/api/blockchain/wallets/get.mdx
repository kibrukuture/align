# Get Blockchain Wallet

Retrieve wallet details including balances for a specific blockchain wallet.

## Method Signature

```typescript
align.blockchain.wallets.get(address: string): Promise<WalletBalance>
```

## Example

```typescript
const wallet = await align.blockchain.wallets.get(
  "0x742d35Cc6634C0532925a3b844Bc9e7595f0aB42"
);

console.log(`Address: ${wallet.address}`);
console.log(`ETH Balance: ${wallet.nativeBalance}`);
```
