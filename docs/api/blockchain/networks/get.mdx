# Get Network

Retrieve configuration details for a specific blockchain network.

## Method Signature

```typescript
align.blockchain.networks.get(network: string): Promise<NetworkConfig>
```

## Example

```typescript
const network = await align.blockchain.networks.get("polygon");

console.log(`Chain ID: ${network.chainId}`);
console.log(`RPC URL: ${network.rpcUrl}`);
console.log(`Explorer: ${network.explorerUrl}`);
```

## Supported Networks

| Network  | Chain ID | Currency |
| -------- | -------- | -------- |
| ethereum | 1        | ETH      |
| polygon  | 137      | MATIC    |
| base     | 8453     | ETH      |
| arbitrum | 42161    | ETH      |
