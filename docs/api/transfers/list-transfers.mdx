# List Transfers

Retrieve all transfers associated with a specific customer.

## Method Signature

```typescript
align.transfers.list(customerId: string): Promise<TransferListResponse>
```

## Parameters

<ParamField path="customerId" type="string" required>
  The unique customer identifier (UUID format)
</ParamField>

## Returns

<ResponseField name="items" type="Transfer[]">
  Array of transfer objects
</ResponseField>

## Examples

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    import Align from "@tolbel/align";

    const align = new Align({
      apiKey: process.env.ALIGN_API_KEY!,
      environment: "sandbox",
    });

    const response = await align.transfers.list(
      "123e4567-e89b-12d3-a456-426614174000"
    );

    console.log(`Total transfers: ${response.items.length}`);

    for (const transfer of response.items) {
      console.log(`${transfer.transfer_id}: ${transfer.status}`);
    }
    ```

  </Tab>
  <Tab title="JavaScript">
    ```javascript
    const response = await align.transfers.list(
      "123e4567-e89b-12d3-a456-426614174000"
    );

    response.items.forEach((transfer) => {
      console.log(`${transfer.transfer_id}: ${transfer.status}`);
    });
    ```

  </Tab>
</Tabs>

### Filter by Status

```typescript
const response = await align.transfers.list(customerId);

const completed = response.items.filter((t) => t.status === "completed");
const pending = response.items.filter((t) => t.status === "pending");

console.log(`Completed: ${completed.length}`);
console.log(`Pending: ${pending.length}`);
```

### Calculate Total Volume

```typescript
const response = await align.transfers.list(customerId);

const totalVolume = response.items
  .filter((t) => t.status === "completed")
  .reduce((sum, t) => sum + parseFloat(t.source_amount), 0);

console.log(`Total volume: $${totalVolume.toFixed(2)}`);
```
