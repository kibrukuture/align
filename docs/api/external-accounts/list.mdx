# List External Accounts

Retrieve all external bank accounts linked to a customer's profile.

## Method Signature

```typescript
align.externalAccounts.list(
  customerId: string
): Promise<ExternalAccountListResponse>
```

## Example

```typescript
const response = await align.externalAccounts.list(customerId);

for (const account of response.items) {
  console.log(`${account.external_account_id}: ${account.type}`);
  console.log(`  Holder: ${account.account_holder_name}`);
}
```
